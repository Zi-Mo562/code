# Skill: Bug Fix

Goal:
Locate and fix a bug with minimal changes.

Workflow:

1. Identify related files.
2. Inspect implementation.
3. Determine root cause.
4. Apply minimal fix.
5. Suggest validation command.

Rules:

- Modify only necessary files.
- Do not refactor unrelated code.

Output format:

Root Cause
Modified Files
Validation Command
Suggested Commit Message
