# Skill: Analyze Project

Goal:
Understand the repository structure and identify key components.

Workflow:

1. Inspect the repository root.
2. Identify project type:
   - Python
   - Node.js
   - C++
   - Embedded
3. Locate important files:
   - entry points
   - configuration files
   - build scripts
4. Identify dependencies.
5. Suggest validation or run commands.

Output format:

Project Type
Key Directories
Entry Points
Validation Commands
