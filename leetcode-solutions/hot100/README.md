# Hot 100 Topics

Use the folders below to group problems by topic.

- `01-array`: arrays, prefix sum, interval basics
- `02-hash`: hashmap, hashset, counting
- `03-two-pointers`: collision pointers, fast/slow pointers
- `04-sliding-window`: substring and window maintenance
- `05-stack-queue`: monotonic stack, deque, bracket problems
- `06-linked-list`: reverse, merge, cycle, pointer operations
- `07-binary-tree`: traversal, BST, path, recursion
- `08-graph`: BFS, DFS, topological style problems
- `09-backtracking`: permutations, combinations, subsets
- `10-binary-search`: answer search and boundary search
- `11-heap`: priority queue, top k
- `12-greedy`: local optimal strategies
- `13-dp`: linear DP, knapsack, state transition
- `14-matrix`: simulation, rotation, search in matrix
- `15-bit-manipulation`: XOR, bit mask

For a study checklist, see `INDEX.md`.
